# Sistema de Registro con Manejo de Eventos

## Descripción
Este proyecto es un sistema de registro básico desarrollado en PHP que permite capturar información de un usuario mediante un formulario web.

## Funcionalidades
- Registro de nombre, correo y edad
- Manejo de eventos usando botones
- Envío de datos con método POST
- Validación del lado del servidor

## Eventos manejados
- Guardar información
- Validar datos
- Limpiar formulario

## Validaciones aplicadas
- Campos obligatorios
- Formato válido de correo
- Edad mayor a cero

## Tecnologías utilizadas
- PHP
- HTML
- CSS básico
